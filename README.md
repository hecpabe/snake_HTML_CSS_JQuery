# Snake en HTML, CSS y JQuery
Juego de Snake que te permite modificar los siguientes parámetros antes de jugar:
- Ancho de Mapa (5-25)
- Alto de Mapa (5-25)
- Tamaño inicial de la serpiente (1-3)
- Velocidad Inicial de la Serpiente (1-4)
